# Movie Searching App using IMDb API and Local Storage

## Overview

This Movie Searching App is a web application that allows users to search for movies using the IMDb API. Additionally, it provides features to view detailed information about each movie, maintain a wishlist, and copy IMDb IDs to the clipboard for easy sharing. The wishlist functionality utilizes the browser's local storage to persist user preferences.

## Features

- **Movie Search:** Utilizes the IMDb API to search for movies based on their titles.
- **Movie Details:** Provides a detailed view of each movie, including information such as the release year and IMDb rating.
- **Wishlist Management:** Allows users to add movies to their wishlist and remove them as needed. The wishlist is stored in the browser's local storage.
- **Copy to Clipboard:** Provides a convenient way to copy IMDb IDs to the clipboard.

## Technologies Used

- **HTML:** For the structure of the web pages.
- **CSS:** For styling and layout.
- **JavaScript:** To implement dynamic and interactive features.
- **IMDb API:** To fetch movie data.
- **Font Awesome:** For the clipboard icon.

## How to Use

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/Noor-Ahmed-12/Movie-Searching-App-using-IMDBI-API-Local-Storage.git
   ```

2. **Open `index.html` in your Web Browser:**

   ```bash
   cd Movie-Searching-App-using-IMDBI-API-Local-Storage
   ```

   Open the `index.html` file in your preferred web browser.

3. **Search for Movies:**

   Use the search bar to find movies based on their titles.

4. **View Movie Details:**

   Click on a movie to view detailed information, including the release year and IMDb rating.

5. **Manage Your Wishlist:**

   - Add movies to your wishlist by clicking the "Add to Wishlist" button.
   - Remove movies from your wishlist by clicking the "Remove from Wishlist" button.

6. **Copy IMDb IDs:**

   - Click on the IMDb ID to copy it to the clipboard.

## Installation

No installation is required. Simply open the `index.html` file in a web browser.
